# sistem-absensi-mahasiswa
Aplikasi manajemen absensi mahasiswa berbasis desktop yang dirancang menggunakan **Arsitektur MVC (Model-View-Controller)** untuk memenuhi Tugas Besar mata kuliah Pemrograman Berorientasi Objek (PBO) Semester 4, Informatika UPN "Veteran" Yogyakarta.

Aplikasi ini mendigitalisasi pencatatan kehadiran kelas dengan memisahkan hak akses antara **Mahasiswa** (Presensi Mandiri) dan **Asisten Dosen** (Panel Manajemen Data & Verifikasi Keterlambatan) secara real-time.
